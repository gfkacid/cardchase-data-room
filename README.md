# CardChase Data Room

Diligence materials for CardChase.

## Contents

### Pricing validation

Near Mint prices from eBay sold listings vs TCGplayer market price. Pikachu ex 276: 486 NM sales over 26 days, 8.6% gap vs TCGplayer, 1.17x ratio.

### Data quality

How raw eBay sold listings become prices we publish. Five filters before a sale moves a price; about a third of listings drop at the identity check.

### eBay seller TAM

Pokémon TCG seller TAM from catalog-matched eBay sold comps. Mar–May 2026: **~$32M** GMV, **~58k** sellers, **~214k** transactions, **2,924** cards, **145** sets.
